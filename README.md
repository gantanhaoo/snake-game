# Snake Game / 贪吃蛇游戏

A classic Snake game built with HTML5 Canvas and JavaScript.

## How to Play / 如何游戏

1. Open `index.html` in your browser
2. Click "开始游戏" (Start Game) button
3. Use arrow keys or WASD to control the snake
4. Eat the red food to grow and earn points
5. Avoid hitting the walls or yourself

## Controls / 控制

- **Arrow Keys / WASD**: Move the snake
- **Space**: Pause/Resume game

## Features / 特性

- Smooth animations
- Score tracking
- High score saved locally
- Speed increases as you score
- Responsive design

## Demo

Simply open `index.html` in any modern web browser to play!
