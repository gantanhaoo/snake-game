# 🚀 5分钟快速注册域名指南

## 已为你准备的域名：`snakegame2024.tk`

所有代码已配置完成，你只需要注册这个域名即可！

---

## 📝 注册步骤（只需3步）

### 步骤 1：访问并搜索域名（1分钟）

1. 打开浏览器，访问：**https://www.freenom.com**
   - ⚠️ 如果无法访问，可能需要使用 VPN（选择美国节点）

2. 在页面顶部的搜索框输入：**`snakegame2024`**

3. 点击搜索或按回车

4. 如果 `snakegame2024.tk` 不可用，尝试：
   - `snakegame2024.ml`
   - `snakegame2024.ga`
   - `snakegame2024.cf`

### 步骤 2：添加到购物车并注册（2分钟）

1. 找到可用的域名，点击 **"Get it now!"** 或 **"添加到购物车"**

2. 选择注册期限：**12个月（免费）**

3. 点击 **"Continue"** 或 **"继续"**

4. 如果还没有账户：
   - 点击 **"Create Account"** 或 **"创建账户"**
   - 填写邮箱、密码、姓名等信息
   - 完成邮箱验证

5. 如果已有账户，直接登录

6. 完成注册流程

### 步骤 3：配置 DNS（2分钟）

注册成功后：

1. 登录 Freenom，进入 **"我的域名"**（My Domains）

2. 找到你的域名，点击 **"管理域名"**（Manage Domain）

3. 进入 **"管理 Freenom DNS"**（Manage Freenom DNS）

4. 添加 **CNAME 记录**：
   - **名称**：`@`（或留空）
   - **类型**：`CNAME`
   - **目标**：`gantanhaoo.github.io`
   - **TTL**：3600

5. 点击 **"Save"** 保存

---

## ⚙️ 在 GitHub 配置（1分钟）

1. 登录 GitHub，进入仓库：**gantanhaoo/snake-game**

2. 点击 **Settings** → **Pages**

3. 在 **Custom domain** 输入：**`snakegame2024.tk`**（或你注册的域名）

4. 勾选 **Enforce HTTPS**

5. 点击 **Save**

---
