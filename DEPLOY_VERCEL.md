---

## 📝 部署步骤

### 步骤 1：注册 Vercel 账户（1分钟）

1. 访问：https://vercel.com
2. 点击 **"Sign Up"** 或 **"登录"**
3. 选择 **"Continue with GitHub"**（使用 GitHub 账户登录，推荐）
4. 授权 Vercel 访问你的 GitHub 账户

### 步骤 2：部署项目（2分钟）

1. 登录 Vercel 后，点击 **"Add New..."** → **"Project"**

2. 在项目列表中，找到 **`snake-game`** 仓库

3. 点击 **"Import"**

4. 配置项目（通常使用默认设置即可）：
   - **Framework Preset**: Other
   - **Root Directory**: `./`（默认）
   - **Build Command**: 留空（静态网站不需要构建）
   - **Output Directory**: `./`（默认）

5. 点击 **"Deploy"**

6. 等待部署完成（通常 1-2 分钟）

### 步骤 3：获取域名（自动）

部署完成后，你会获得一个域名，格式类似：
- `snake-game-xxx.vercel.app`
- 或 `snake-game-git-master-xxx.vercel.app`

### 步骤 4：测试（1分钟）

1. 点击部署后的链接，在浏览器中打开
2. 确认游戏正常运行
3. **在微信中打开链接，测试是否能正常访问**

---

## 🎯 自定义域名（可选）

如果你想使用自定义域名：

1. 在 Vercel 项目页面，点击 **"Settings"** → **"Domains"**
2. 输入你的域名（如：`snake.yourdomain.com`）
3. 按照提示配置 DNS 记录
4. 等待 DNS 生效

---

## 📱 在微信中测试

部署完成后：

1. 复制 Vercel 提供的域名（如：`https://snake-game-xxx.vercel.app`）
2. 在微信中打开这个链接
3. 应该能正常访问，不会被拦截！

## 🆘 遇到问题？

### 问题1：找不到仓库
- **解决**：确保 Vercel 已授权访问你的 GitHub 账户
- 在 Vercel 设置中重新连接 GitHub

### 问题2：部署失败
- **解决**：检查项目结构，确保 `index.html` 在根目录
- 查看 Vercel 的部署日志

### 问题3：微信中仍然无法访问
- **解决**：尝试使用自定义域名
